## 📌 Présentation

Le site web du **Cabinet Dentaire Tissir** est conçu pour offrir une expérience utilisateur fluide et rassurante. Il met en avant l'expertise du cabinet à travers un design élégant, une navigation intuitive et des fonctionnalités avancées pour faciliter la prise de contact et la consultation.

---

## ✨ Fonctionnalités Clés

- 🌍 **Multilingue & RTL** : Support complet du Français et de l'Arabe avec adaptation automatique de la mise en page (Right-to-Left).
- 📹 **Consultation Vidéo** : Assistant de réservation en 3 étapes pour des téléconsultations simplifiées.
- 🖼️ **Galerie Avant/Après** : Vitrine des résultats cliniques démontrant la qualité des soins esthétiques.
- 📱 **Design Responsive** : Optimisé pour tous les supports (Mobile, Tablette, Desktop).
- 🚨 **Gestion des Urgences** : Accès rapide aux informations de contact prioritaires.
- 🎨 **Esthétique Premium** : Utilisation d'une palette de couleurs harmonieuse, de typographies modernes et de micro-animations.

## 🏗️ Architecture du Projet

```text
src/
```

## 🚀 Installation et Démarrage

### Prérequis

- [Node.js](https://nodejs.org/) (version 18 ou supérieure recommandée)
- [npm](https://www.npmjs.com/)

### Étapes

1. **Cloner le dépôt**
   ```bash
   git clone https://github.com/ridamellouki3/Cabinet-Tissir.git
   cd Cabinet-Tissir/frontend
   ```

2. **Installer les dépendances**
   ```bash
   npm install
   ```

3. **Lancer le serveur de développement**
   ```bash
   npm run dev
   ```

4. **Bâtir pour la production**
   ```bash
   npm run build
   ```
